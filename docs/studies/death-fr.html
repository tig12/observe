Deaths in France 1970 - 2025

This study is based on data from www.data.gouv.fr/datasets/fichier-des-personnes-decedees.
It contains 28 803 835 persons, with birth day and death day.
The purpose of this study is to observe if statistical anomalies appear when observing the angle between the planets at birth and death.

Execution

Build sqlite database

Text files downloaded from data.gouv.fr are first loaded in a local sqlite database, death-fr.sqlite3 (with an index on birth days).
It contains one table, person with this structure: 
create table person(
    fname varchar(80),
    gname varchar(80),
    sex character(1),
    bday character(8),
    bcode character(5),
    bname character(30),
    bcountry varchar(80),
    dday character(8),
    dcode character(5),
    dact character(9)
);
create index idx_bday ON person(bday);
This is done by another program, g5: https://github.com/tig12/g5. 
php run-g5.php enrich deathfr raw2sqlite 1970-2025 > data/tmp/enrich/death-fr/sqlite-build-report.log

Compute planets

A local sqlite database, planets.sqlite3 contains the positions of planets computed at 12:00, for each day between 1700 and 2030.
Generated by: 
php run-observe.php prepare planets 1700-2030

Compute splits
