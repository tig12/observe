Gauquelin heredity data

The data used for this study originates in a work of Nick Kollestrom, who assembled the Gauquelin heredity data in a single file and published it on newalchemypress.com.
This file contains 20 916 birth dates of mother, father and child.
One of the interests of these data is that the birth time is provided.
This dataset has not been processed yet by the 2026 version of the program (in particular control groups have not been computed), so do not contain any statistical analysis.

The raw data

The departure point of this study is a PDF file downloadable from https://newalchemypress.com/gauquelin/gauquelin_docs/mom_dad_kid_final20916_3a-m_column_first.pdf.

Details about this file and its conversion to a CSV file can be found on github.com/tig12/g5-other/tree/main/gauquelin/heredity-newalch-20916.
