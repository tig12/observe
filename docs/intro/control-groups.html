Control groups

The distributions computed from the original data set are called the observed distributions. The next step is to build the expected distributions. The comparison between observed and expected distribution will permit to see if the observed distributions show anomalies.
In this program, the expected distributions are built with control groups.

Definition

A control group is a fictional group built from the observed group.
Here, it is important to note that precautions need to be taken, and that the way control groups are built implies arbitrary choices.
For example, to build a control group of a data set containing birth and death dates, the simplest idea would be to:
  • take the birth date of a record,
  • randomly select an other record,
  • build a record of the control group with the birth date of the first record and the death date of the second record.
Obviously, a precaution is needed, because the birth date of the newly created record must be anterior to its death date.

And other questions arise: Do we want to keep properties of the observed group ? For example, do we want that the control groups respect the seasonal distribution of births ? Do we want to respect the age at death distribution ?

The answers to these questions imply arbitrary choices which must be identify. These answer affect the way we compute the control groups and may affect the comparison with the expected distributions.

Expected distributions

To build the expected distributions, several control groups are computed, and the average distributions of these controls are computed.
