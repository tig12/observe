Studies

In the context of this program, a study is based on a dataset containing dates.
From these data, a set of computations are done, leading to the statistical tests:
init
splitBuild the sub-groups
observedCompute the distributions for each split
controlBuild the control groups
expectedBuild the expected distributions from control groups
statsCompute statistical informations about the distributions
outputGenerate a presentation of the results
(orange : computations specific to each study - green : computations common to all studies)
These computations are done issuing commands with the CLI run-observe.php (CLI = Command Line Inerface, a tool used in a console).
The intermediate data generated by the commands are stored in the working directory of the study.
Except for command output, which generates its results in the output directory.

Splits

We often need to divide the original dataset in different subgroups to see if they show different distributions.
For example, how do the distributions of babies who died a few days after their birth compare with the distributions of general population ?
In this program, this is handled by the notion of split.
A split permits to generate subgroups from the original groups.
In general, a study has a split with only one subgroup containing the whole dataset.
