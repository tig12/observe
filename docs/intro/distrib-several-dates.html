Several dates distributions

When a dataset contains several dates (for example birth and death dates), the program can compute the single date distributions and other distributions generated by the relations between the data of each date.
This page lists the distributions handled by the program.
The examples come from the a00 study, a dataset containing mother, father and child bith dates + wedding (MFCW) dates.

Date differences

Here are examples of distributions built computing the differences between two dates.

Inter-aspects

"Inter-aspect" is the name used in this program to designate an aspect (an angle) between planets generated by two dates.
For example:
  • The longitude of venus at the moment of father's birth is 100°
  • The longitude of mars at the moment of mother's birth is 150°
  • The we say that the inter-aspect between venus' mother and mars' father is 50°

Summary

The distributions generated by a data set with several dates include the distributions coming from each individual date (as described in page single date distributions) plus distributions coming from the relations between 2 dates.
Taking the example of a data set containing birth and death dates, the following hierarchy of files is generated: 
distributions
    ├── birth
    │   ├── aspects
    │   │   ├── JU-NE.csv
    │   │   ├── ...
    │   │   └── VE-UR.csv
    │   ├── planets
    │   │   ├── JU.csv
    │   │   ├── ...
    │   │   └── UR.csv
    │   ├── day.csv
    │   └── year.csv
    ├── birth-death
    │   ├── interaspects
    │   │   ├── JU-JU.csv
    │   │   ├── ...
    │   │   └── VE-VE.csv
    │   └── age.csv
    └── death
        ├── aspects
        │   ├── JU-NE.csv
        │   ├── ...
        │   └── VE-UR.csv
        ├── planets
        │   ├── JU.csv
        │   ├── ...
        │   └── UR.csv
        ├── day.csv
        └── year.csv
If the data set contains more than two dates, the program only handles relations between 2 dates.
For example, for a dataset containing mother, father and child bith dates + wedding (abbreviated M, F, C and W), the main directories will be: 
distributions
    ├── C
    ├── C-W
    ├── F
    ├── F-C
    ├── F-W
    ├── M
    ├── M-C
    ├── M-F
    ├── M-W
    └── W
This is a limitation of the program, but many other distributions can be imagined, involving relations between three or more dates.
